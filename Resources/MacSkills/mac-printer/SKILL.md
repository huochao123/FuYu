# 打印机与扫描仪

1. 确认设备型号、连接方式、网络地址、耗材状态和具体错误。
2. 区分设备离线、队列卡住、驱动问题、AirPrint发现失败和应用文档问题。
3. 先暂停或取消单个失败任务，不立即重置整个打印系统。
4. 驱动只从系统更新或厂商官方来源获取。
5. 重置打印系统会移除所有队列和预设，必须由用户确认。
6. 用测试页或小型扫描任务验证，而非只看设备显示在线。
