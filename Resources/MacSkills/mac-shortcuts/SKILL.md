# 快捷指令与自动化

1. 明确触发方式、输入、输出、目标应用、失败处理和完成标准。
2. 优先使用系统快捷指令与公开接口；需要脚本时限制权限和作用范围。
3. 自动删除、发送、购买、发布或修改安全设置必须保留人工确认。
4. 处理文件时先预览、避免覆盖并提供撤销或日志。
5. 辅助功能、自动化和文件权限由用户在系统设置中明确授予。
6. 用测试数据运行并验证幂等性，避免重复触发造成重复操作。
