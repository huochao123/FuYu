# 浏览器性能与隐私

1. 区分单个网站、标签页数量、扩展、用户配置和浏览器整体故障。
2. 检查CPU、内存、能耗与页面时间，避免把所有子进程视为不同应用。
3. 清缓存不应作为万能修复；先说明可能退出登录或重新下载资源。
4. 逐个禁用可疑扩展并复测，不直接删除书签、密码或配置文件。
5. 摄像头、麦克风、通知和下载权限按网站核对。
6. 修复后用原网站和相同步骤验证性能与功能。
