# iCloud与云盘同步

1. 确认Apple账户、iCloud Drive开关、网络、存储配额和具体文件状态。
2. 区分已下载文件、云端占位文件、正在上传、同步冲突与本地副本。
3. 不把云朵图标或“优化Mac存储”误认为文件丢失。
4. 移动桌面与文稿、退出账户或删除云端文件前先说明跨设备影响。
5. 保留冲突副本，禁止自动覆盖内容不同的同名文件。
6. 处理后从另一设备或网页端验证云端状态。
