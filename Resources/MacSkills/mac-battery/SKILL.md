# 电池与电源健康

1. 读取电源来源、充电状态、最大容量、循环次数和系统给出的维修状态。
2. 将电池自然老化与应用异常耗电分开，不用单次掉电速度判断健康。
3. 分析持续高能耗进程、屏幕、外设、网络与睡眠唤醒记录。
4. 说明续航影响和证据可信度；不要承诺软件能修复电芯老化。
5. 调整设置前说明体验代价，维修或更换只能由用户决定。
6. 优化后用一段时间内的真实耗电趋势复查。
