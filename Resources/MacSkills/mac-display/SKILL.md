# 显示器与图形输出

1. 确认显示器型号、连接方式、线缆/扩展坞、分辨率、刷新率和HDR状态。
2. 区分无信号、系统未识别、错误模式、闪屏和应用层渲染问题。
3. 高分辨率高刷新率受接口带宽限制，先核对链路能力再改设置。
4. 变更分辨率、色彩配置或HDR属于可逆操作，但要保留原值。
5. 外接屏故障交叉测试端口、线缆和直连，避免只靠重启判断。
6. 花屏或持续黑屏伴随系统崩溃时升级检查硬件与panic记录。
