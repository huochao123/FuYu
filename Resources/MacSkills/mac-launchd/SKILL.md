# 启动项与后台服务

1. 同时检查登录项、后台项目、用户LaunchAgents和系统LaunchDaemons。
2. 识别签名、来源、路径、当前状态和最近错误。
3. 不因名称陌生就删除plist；先说明用途和影响。
4. 停用、卸载或删除必须确认并保留恢复方式。
5. 修改后用真实launchd状态和日志验证。
