# 蓝牙设备诊断

1. 确认设备电量、配对状态、当前连接主机、距离和具体故障时间。
2. 区分无法发现、配对失败、随机断连、输入延迟和音频质量下降。
3. 蓝牙耳机通话音质下降可能来自麦克风配置切换，不等于耳机损坏。
4. 先重连单个设备并排除多主机抢占和2.4GHz干扰。
5. 忘记设备或重置配置会影响现有配对，执行前必须说明。
6. 修复后持续观察连接和输入/音频状态，而非只看已连接图标。
