# 应用安装卸载与残留

1. 确认应用官方来源、签名、架构、最低系统版本和安装形式。
2. 区分拖入Applications的应用、PKG组件、后台助手、系统扩展和用户数据。
3. 升级前保留配置和数据，验证新版能启动后再清理旧版本。
4. 卸载不能只删应用图标；先列出相关组件并区分缓存与用户数据。
5. 删除、停用后台项或系统扩展必须确认并保留恢复办法。
6. 完成后验证进程、登录项和版本，只保留用户要求的最新版。
