# 隐私权限与系统安全

1. 读取当前授权状态并指出准确的系统设置入口。
2. 区分未请求、已拒绝、已授权和系统限制。
3. 不绕过TCC、SIP或Gatekeeper，不静默修改安全设置。
4. 需要用户点击或认证时明确标记“只能由用户完成”。
5. 权限变化后重新验证真实能力，不能仅凭界面按钮宣布成功。
